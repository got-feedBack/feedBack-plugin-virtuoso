Virtuoso

Virtuoso generates guitar/bass practice routines — scales, arpeggios, chords, rhythm and technique drills, timed workouts, and jam backing — and plays them back inside the plugin with its own transport, audio engine, and pitch-based scoring.

Renderers

Four playback surfaces: the host's 3D Highway (borrowed via window.slopsmithViz_highway_3d) plus built-in 2D Highway, Tab, and Notation renderers. Playback is fully contained in the plugin — it never launches the main FeedBack player.

Data

Saved presets and custom tunings are stored in FeedBack's shared metadata database. A legacy plugin_data/virtuoso/presets.json file, if present, is migrated in automatically on first load.

Save folder for cards

Where the Save button (and auto-save, if on) writes the results / copy card PNG. Leave blank to use your Pictures folder (e.g. C:\Users\you\Pictures). Must be an absolute path. Shared with note_detect — both plugins write their cards to this folder.
