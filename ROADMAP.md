# Virtuoso ‚Äî Roadmap

> **Virtuoso is a practice-studio plugin for the FeedBack host** ‚Äî it generates guitar/bass (piano/drums in scope) practice routines and plays them back inside the plugin. It relaunched at **v0.1.0** as a clean-history rebrand of a mature codebase (formerly developed under another name; the full pre-0.1.0 development history is preserved in the original private archive repo). This file is the authoritative **"shipped vs planned"** record and the cross-session handoff log ‚Äî **read it at session start.**

> Architecture + conventions live in `CLAUDE.md` (the constitution) and `docs/`; the user-facing feature list is in `README.md`. Don't duplicate those here.

## Where this is (v0.1.0)

Virtuoso ships a mature, working feature set inherited at relaunch ‚Äî **this is not a greenfield 0.1.0**, it's an established plugin renamed and re-homed:

- **Four-mode DAW shell** ‚Äî **Ladder** (curated skill pathways), **Custom** (full manual config), **Workout** (timed multi-block sessions), **Jam** (play-along over a procedural backing band). One shared shell; **contained playback** (Virtuoso owns its own transport/clock/Web-Audio + pitch tracker ‚Äî no host-player handoff).
- **Generation engine** ‚Äî the host- and DOM-independent core (`screen.js` ¬ß1‚Äì¬ß10): scales/modes, arpeggios, chords, rhythm/technique drills, riff-vocabulary, CAGED/3NPS/Open shapes, the voicing engine, and the procedural backing band (29 genre styles via `STYLE_PALETTES`‚Üí`ARRANGEMENT_RECIPES`). The **chart** is the portable artifact.
- **Four renderers** ‚Äî 3D Highway (borrowed host viz), 2D Highway, Tab, Notation.
- **Grading** ‚Äî consumes the host **note_detect** verifier (the *consume-the-host-judge* rule ‚Äî no parallel grader). Plus hand-marks, the proof-loop, XP/leveling/badges, custom tunings, sample-bank instruments, and the amp/cab "Rig".
- **Audio** ‚Äî self-hosted WebAudioFont GM sampler + sampled electric-DI + in-browser amp/cab IRs under a master limiter; optional host `nam_tone` borrow for the live input tone.

Verification: 18 Playwright smoke suites (`.claude/skills/run-virtuoso/`, `npm test`) + the in-`screen.js` startup guards. **Smoke 18/18 green at relaunch.**
